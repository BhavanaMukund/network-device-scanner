# Network Device Scanner

This project is a Layer 2 network scanning tool built using Python and Scapy.

## Features
- ARP based host discovery
- IP – MAC address mapping
- Works in local subnet
- Flask web interface for output

## Tools Used
- Python
- Scapy
- Flask
- Kali Linux
- HTML

## Description
The scanner sends ARP broadcast packets in the local network and collects responses to identify active devices.

This project was created for learning network reconnaissance and packet crafting.
